# Jump Shinobi

Jump Shinobi is a fast-paced 2D action-platformer developed in C++ using the Qt framework. Players take control of a skilled shinobi and traverse increasingly difficult levels filled with obstacles, enemies, and platforming challenges. The game combines responsive movement mechanics, collision detection, enemy interactions, and score tracking to create an engaging arcade-style experience.

## Features

- Physics-based jumping and movement system
- Real-time collision detection
- Enemy AI and combat interactions
- Sprite-based animations
- Multi-level progression
- Score and life management
- Object-oriented game architecture
- Event-driven gameplay using Qt
- Efficient resource and asset management

## Technologies Used

- C++
- Qt Framework
- Qt Graphics View Framework
- Object-Oriented Programming (OOP)
- Event-Driven Architecture
- Game Loop Design Patterns

## Gameplay

The objective is to guide the shinobi through each level while avoiding hazards, defeating enemies, and reaching the goal. Players must utilize precise timing, movement, and platforming skills to overcome increasingly challenging environments.

## Project Architecture

The project is structured around several core systems:

- **Player System** – Handles movement, jumping, health, and interactions.
- **Enemy System** – Controls enemy behavior and AI logic.
- **Collision Engine** – Detects and resolves object interactions.
- **Level Manager** – Loads and manages game stages.
- **Rendering Engine** – Displays sprites, animations, and game objects.
- **Score Manager** – Tracks player progress and achievements.
